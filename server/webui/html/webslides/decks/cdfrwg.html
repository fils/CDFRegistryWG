In Conclusion

EarthCube CDF RWG is working on patterns for the self hosting and publishing of semantic facility metadata.

Self Hosted

Facilities can self host and manage locally their semantic metadata. Any number of consumers can then harvest and value add to that.

* * *

Web Architecture

Both the publishing and consumption of the semantic metadata follows established web architecture approaches

Adaptable Pattern

The pattern is scalable and adaptable to a range of applications, such as data.

* * *

Future work

ESSO has funded work into exploring extension of this pattern to data sets. Google is also pursuing this pattern.
